Flexr is a responsive grid for those who simply want a grid. Using only CSS, columns react in three ways: expand, shrink, and stretch.

Why Flexr Grid?

Flexr offers a super lightweight framework, a pure focus on the responsive grid, better breakpoints for smarter layouts, easy implementation methods, gutters that adapt to screen sizes, and you don't waste time overriding code or writing extra classes.
