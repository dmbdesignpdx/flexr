Defining Columns

Shrinking Columns is the default type of collapsing, so there is no need to declare any class for it.

To set Expanding Columns, all you need to do is declare the expanding class in the row parent:

<div class="row exp">

  <div class="col-4">

  <div class="col-4">

  <div class="col-4">

</div>

Justifying Columns

Justifying columns is as easy as declaring what you want in the row parent. For example, right-justification:

<div class="row jus-r">

  <div class="col-x">

</div>

If you wanted to split columns, declare which direction you want to split based on the row parent's justification:

<div class="row">

  <div class="col-x spl-r">

</div>
