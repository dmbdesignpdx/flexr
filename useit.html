One Class Will Do

Creating a column requires only one class. Just decide how big of a column you want (1–11, no need to create a column size 12, just use the row!), and build:

<div class="row">

   <div class="col-6">

   <div class="col-6">

</div>

Defining Columns

Shrinking Columns is the default type of collapsing, so you do not have to declare any class for it.

To set Expanding Columns, all you need to do is declare the expanding class in the row:

<div class="row exp">

   <div class="col-4">

   <div class="col-4">

   <div class="col-4">

</div>

Justifying Columns

Justifying columns is as easy as declaring what you want in the row. For example, right-justification:

<div class="row jus-r">

   <div class="col-x">

</div>
