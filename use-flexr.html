Flexr Documentation

First Thing's First

You gotta link the Flexr Grid CSS.

<head>

   <link href="path-to/flexr.css" rel="stylesheet">

   <link href="path-to/your.css" rel="stylesheet">

</head>

Setting Up

Here's the basic structure:

<nav | header | section | footer>

    <row>

        <column(s)>

            your stuff!

        </column(s)>

    </row>

</nav | header | section | footer>

Flexr works best when the row's parent has a width set to 100%.

One Class Will Do

Creating a column requires only one class col- defined by its size [1–12] (total column size no more than 12 per row):

<div class="row">

   <div class="col-6">...</div>

   <div class="col-6">...</div>

</div>

Defining Columns

Shrinking Columns is the default type of collapsing.

To define Expanding Columns, all you need to do is add the expanding class exp in the parent row:

<div class="row exp">

   <div class="col-3">...</div>

   <div class="col-3">...</div>

   <div class="col-3">...</div>

   <div class="col-3">...</div>

</div>

See the difference in behavior here.

Justifying Columns

The default justification is centered. To justify the columns to left or the right, just add the class jus-[l,r] to the parent row. For example, left-justification:

<div class="row jus-l">

   <div class="col-4">...</div>

</div>

Flexr Nav Objects

You can use Flexr Nav Objects to help keep the logo and menu links inline with the Flexr grid. The logo- and menu- objects act just like columns and each object has a size that fits into the grid [s,m,l,xl].

The contents of the menu object will be automatically spaced evenly. If that's not what you want, you can override that by using the same justification classes previously mentioned.

Here is an example of a nav utilizing the Flexr options:

<nav class="row">

    <div class="logo-m">...</div>

    <ul class="menu-xl">...</ul>

</nav>
