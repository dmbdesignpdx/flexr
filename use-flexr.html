Flexr Documentation

First Thing's First

You gotta link the Flexr Grid CSS before your CSS.

<head>

...

<link href=".../flexr.css" rel="stylesheet">

<link href=".../your.css" rel="stylesheet">

...

</head>

Setting Up

Here's the basic structure:

<section | container>

<row>

    <column(s)>

        your stuff!

    </column(s)>

</row>

</section | container>

This structure isn't set in stone, you may choose to use a section or container as a row. It's up to you!

One Class Will Do

Creating a column requires only one class col- defined by its size [1–12] (total column size no more than 12 per row):

<div class="row">

<div class="col-6">...</div>

<div class="col-6">...</div>

</div>

If you'd like to use a column with a width that you define (i.e. for a logo), simply give the column the class col-x and define the width in your css.

Defining Columns

Shrinking Columns is the default type of collapsing.

To define Expanding Columns, all you need to do is add the expanding class exp in the parent row:

<div class="row exp">

<div class="col-3">...</div>

<div class="col-3">...</div>

<div class="col-3">...</div>

<div class="col-3">...</div>

</div>

To define Filling Columns, just add the filling class fill in the parent row. Filling Columns make their parent row full width to its parent, and divide the space between themselves equally with no gutters.

See the difference in behavior here.

Justifying Columns

The default justification is centered. To justify the columns to left or the right, just add the class jus-[l,r] to the parent row. For example, left-justification:

<div class="row jus-l">

<div class="col-4">...</div>

</div>
