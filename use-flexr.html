Flexr Documentation

First Thing's First

You gotta link Flexr CSS. In your HTML:

<head>

   <link href="path-to/flexr.css" rel="stylesheet">

   <link href="path-to/your.css" rel="stylesheet">

</head>

Or in your CSS:

@import "path-to/flexr.css";

/*  Rest of your CSS  */

One Class Will Do

Creating a column requires only one class col- defined by its size [1–12] (total [num] not exceeding 12):

<div class="row">

   <div class="col-6">...</div>

   <div class="col-6">...</div>

</div>

Defining Columns

Shrinking Columns is the default type of collapsing.

To define Expanding Columns, all you need to do is add the expanding class exp in the parent row (note, it is recomended to use this for rows with columns sizes of 1, 2, and 3):

<div class="row exp">

   <div class="col-3">...</div>

   <div class="col-3">...</div>

   <div class="col-3">...</div>

   <div class="col-3">...</div>

</div>

Justifying Columns

The default justification is centered. To justify the columns to left or the right, just add the class jus-[l,r] to the parent row. For example, left-justification:

<div class="row jus-l">

   <div class="col-4">...</div>

</div>

Navigation Objects

You can use Flexr navigation objects to help keep the logo and menu links inline with the Flexr grid. The logo- and menu- objects act just like columns and each object has a size that fits into the grid [s,m,l,xl]. Note: logo object doesn't have xl size.

The contents of the nav element and the Flexr menu object are automatically spaced evenly. If that's not what you want, you can override that by using the same justification classes previously mentioned.

By default, the logo object will always stay justified to the left. However, if you'd like it to move to the center for the phone, simply add the mtc class.

Here is an example of a nav utilizing the Flexr options:

<nav>

   <div class="row">

      <div class="logo-m">...</div>

      <ul class="menu-xl">...</ul>

   </div>

</nav>
